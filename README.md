# JOGO DE PLATAFORMA 
Jogo estilo plataforma 2D criado utilizando Python e a biblioteca do PgZero.
O jogo é simples e funciona da seguinte forma: Ao iniciar o game você será um personagem que pode pular e andar para os lados e pisar em cima dos inimigos.
Dentro do jogo existe uma física onde você não atravessa as plataformas e nem os inimigos, ao encostar no mesmo você perde uma vida, se perder as 3 o jogo reinicia e você pode tentar novamente.
Existe uma interface simples para poder começar o jogo, deixar no mudo ou sair.
Esse foi um projeto pequeno que quis fazer para treinar um pouco da minha lógica.
COMANDOS: ← ↑ → ↓ ou WASD, e espaço para pular.
